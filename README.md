# 📱 Cell Phone Price Prediction

## 📌 Project Overview
This project predicts the price range of mobile phones using machine learning classification algorithms.

## 📊 Dataset Features
- Battery Power
- RAM
- Internal Memory
- Camera
- Screen Size
- Processor Speed
- 4G / 5G Support

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## 🤖 Models Implemented
- Logistic Regression
- Random Forest
- Decision Tree
- KNN
